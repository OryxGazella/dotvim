#Installation: #

## Clone ##
    git clone git://github.com/oryxgazella/dotvim.git ~/.vim

## Create symlinks: ##

    ln -s ~/.vim/vimrc ~/.vimrc


## Update submodules ##

    cd ~/.vim
    git submodule init
    git submodule update
